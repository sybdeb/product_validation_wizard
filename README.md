# product_validation_wizard
een wizard in Odoo 18 die controleert of een product klaar is voor publicatie (bijv. op je webshop). De checks zijn:  Er moet een foto (image) aanwezig zijn. Er moet een prijs ingesteld zijn (bijv. verkoopprijs > 0). De voorraad bij de goedkoopste leverancier moet minstens x stuks zijn. Het product moet voldoen aan je verkoopcriteria
